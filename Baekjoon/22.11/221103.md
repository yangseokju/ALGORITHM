# 백준 알고리즘 문제
### 11651_좌표 정렬하기 2

문제 : 2차원 평면 위의 점 N개가 주어진다. 좌표를 y좌표가 증가하는 순으로, y좌표가 같으면 x좌표가 증가하는 순서로 정렬한 다음 출력하는 프로그램을 작성하시오.

입력 : 첫째 줄에 점의 개수 N (1 ≤ N ≤ 100,000)이 주어진다. 둘째 줄부터 N개의 줄에는 i번점의 위치 xi와 yi가 주어진다. (-100,000 ≤ xi, yi ≤ 100,000) 좌표는 항상 정수이고, 위치가 같은 두 점은 없다.

*내 제출*
```python
N = int(input())
s = [0] * N

for i in range(N):
    a, b = map(int, input().split())
    s[i] = [a, b]

# 람다함수를 이용해 x좌표를 오름차순으로 정렬해준다.(y좌표가 동일한 경우 x좌표 순으로 정렬해주기 위해서)
s.sort(lambda x:x[0])
# 람다함수를 이용해 y좌표를 오름차순으로 정렬해준다.
s.sort(lambda x:x[1])

for i in range(N):
    print(*s[i])
```
